# Pandas Data Exploration and Cleaning

This is my first assignment where I worked with a real dataset using Python and Pandas.

## What I Did
I took an ecommerce shopping dataset and performed basic data exploration and cleaning on it.

## Dataset
- File used: Combined_dataset.csv
- Original size: 1000 rows, 24 columns
- After cleaning: 1000 rows, 26 columns

## Steps I Followed

1. Loaded the CSV file using pandas
2. Explored the data - checked shape, columns, data types
3. Found and handled missing values
4. Filtered products with rating greater than 4
5. Checked for duplicate rows - none were found
6. Created a new column total_amount = final_price x quantity
7. Saved the cleaned data as cleaned_dataset.csv

## Files in this repo
- data_Exploration_cleaning.ipynb — my jupyter notebook
- Combined_dataset.csv — original dataset
- cleaned_dataset.csv — cleaned output file
