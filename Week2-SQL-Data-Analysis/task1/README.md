# Superstore Sales Analysis Using SQL

## Celebal Technologies Summer Internship – Week 2

### Project Overview

This project focuses on SQL-based data analysis using the Superstore Dataset. The objective is to analyze sales data using filtering, aggregation, sorting, business queries, and data validation techniques.

### Dataset

* Dataset: Superstore Dataset
* Source: Kaggle Superstore Dataset

### Objectives

The following tasks were performed as part of the assignment:

1. Loaded the dataset into a SQL database.
2. Explored table structure and sample records.
3. Applied WHERE filters on:

   * Region
   * Category
   * Date
   * Sales

4. Performed aggregations using GROUP BY:

   * Total Sales
   * Total Quantity
   * Average Sales

5. Sorted and limited results:

   * Top Products
   * Top Categories

6. Solved business use cases:

   * Monthly Sales Trends
   * Top Customers
   * Duplicate Orders

7. Performed data validation:

   * Row Counts
   * Missing Value Checks
   * Duplicate Record Checks

### Key Findings

* Sales performance varies across different regions.
* Technology and Office Supplies contribute significantly to overall sales.
* A small group of customers accounts for a large share of revenue.
* Monthly sales trends indicate fluctuations in customer demand.
* Data quality checks revealed minimal missing values and manageable duplicate records.
