# E-Commerce Database Assessment

## Overview

This project demonstrates the implementation of a relational E-Commerce database using MySQL. The assignment covers database design, constraints, indexing, filtering, aggregation, joins, transactions, and SQL query optimization concepts.

## Project Structure

```text
task2/
│
├── docs/
│   └── theory_answers.md
│
├── sql/

│   ├── sample_data.sql
│   └── sql_query_solutions.sql
│
└── README.md
```

## Files Description

### database_schema.sql

Contains the complete database schema including:

* Customers table
* Products table
* Orders table
* Order Items table
* Primary Keys
* Foreign Keys
* Constraints
* Indexes

### sample_data.sql

Contains sample records inserted into all database tables for testing and analysis.

### sql_query_solutions.sql

Contains SQL solutions for:

* SQL Basics

* Aggregation Functions

* CASE Statements
* Transactions

### theory_answers.md

Contains detailed explanations for theoretical questions related to:

* Primary Keys
* Constraints

* Joins
* Foreign Keys
* ACID Properties
